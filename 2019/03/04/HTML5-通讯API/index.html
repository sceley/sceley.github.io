HTML5 通讯API

  • Web通讯

序言

HTML5之前，前后端之间的数据交互比较单一，只能通过ajax技术实现两端之间的数据交互，而且只能由客户端向服务器端发送请求进而携带着数据发往服务器端，然后等待服务器端响应，响应中携带着客户端需要的服务器端数据，而服务器端无法主动向客户端主动发送数据。HTML5现世后，涌现出了两个新的API, WebSocket、SSE，实现了服务器端主动向客户端发往数据。
